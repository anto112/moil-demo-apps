## Moil Camera Monitoring System

**How to use:**

1. clone this repository

```
$ git clone https://github.com/anto112/moil-demo-apps.git
```

2. install requirement following this command:

```
$ pip install -r requirements.txt
```

3. run the apps:

```
$ python model.py
```
